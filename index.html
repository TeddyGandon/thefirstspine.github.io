TFS
Github NPM

Building online board games
through open source technology

Infrastructure

Monitoring, deployment... all the server/docker-related things that empowers the TFS platform.


boiler

Boiler is a Docker orchestration software built on top of nginx & docker-compose.

Services

The TFS platform is powered by a Server-Oriented Architecture. These services are served by Boiler in containers.


arena

The arena service is the service that will manage all the games & wizzards accounts in the Arena products.


auth

  Protected repository
This repository is only available for The First Spine development team.

Main authentification service. Manages registrations & token generation.


bots

  Protected repository
This repository is only available for The First Spine development team.

Bot service to act like a human in the games. The bots should not be used for ranked games & tournaments.


messaging

Simple realtime messaging system. Contains an API to send message through sockets, where clients can subscribe. It's lighter than Apache Kafka because we have simple needs here.


rest

Rest API. They are all static assets in the API, such as cards, decks, avatars, game types, etc.


rooms

The rooms service will manage discussions between players inside rooms, contained in subjects.


shop

  Protected repository
This repository is only available for The First Spine development team.

Manages all the purshases with real money. Works with the Stripe API.


solid-pancake

Solid Pancakes is an event-based consumer that tracks usages on the TFS products.


website

Website built with Sails.js, using all the other services to serve data to the players.

Tools & assets

All the tools & the assets we need to construct TFS apps.


flame

Data computing & analysis on-demand on CLIs served by Google Fire.


@thefirstspine/certificate-authority

Javascript and Typescript service to validate public keys against private key stored in environment variable.


@thefirstspine/nest-filesocket

This NestJS module allows you to call your app context through local file socket.


@thefirstspine/types-arena

Types definition for Arena resources. Contains all interfaces & build scripts.


@thefirstspine/auth

Javascript & Typescript dependency to help developers to use the auth net service & guard.


@thefirstspine/auth-nest

Built on top of @thefirstspine/auth. Provides Nest services.


@thefirstspine/certificate-validation-nest

Built on top of thefirstspine-certificate-validation. Provides Nest services.


@thefirstspine/messaging

Javascript and Typescript service to interact with the messaging net service.


@thefirstspine/messaging-nest

Built on top of thefirstspine-messaging. Provides Nest services.


@thefirstspine/logs

Javascript and Typescript service to deliver unified logs. Works with Winstonjs.


@thefirstspine/logs-nest

Built on top of thefirstspine-logs. Provides Nest services.


@thefirstspine/types-rest

Types definition for REST resources. Contains all interfaces & build scripts.
