2020 Autumn cleanup

Starting November 6th we will do a complete cleanup of our infrastructure! All the concerned repositories are marked with that icon .

Split Arena service

The arena service will be splitted in two services: game & wizard.

New calendar service

Cycles & events will be moved to a new calendar service.

No more compose

Stop using of compose in Boiler.

True stateless

No more in-memory data, no more volumes.

Infrastructure

Monitoring, deployment... all the server/docker-related things that empowers the TFS platform.


boiler

  Autumn cleanup
This repository will be part of the Autumn cleanup.

Boiler is a Docker orchestration software built with Cobra to ship secure apps. Works along nginx, Docker & certbot.

Services

The TFS platform is powered by a Server-Oriented Architecture. These services are served by Boiler in containers.


arena

  Autumn cleanup
This service will be splitted in two services: games & wizards.

The arena service is the service that will manage all the games & wizzards accounts in the Arena products.


auth

  Autumn cleanup
We planned to not use Docker Compose anymore. The database used will be external.

  Protected repository
This repository is only available for The First Spine development team.

Main authentification service. Manages registrations & token generation.


bots

  Protected repository
This repository is only available for The First Spine development team.

Bot service to act like a human in the games. The bots should not be used for ranked games & tournaments.


messaging

Simple realtime messaging system. Contains an API to send message through sockets, where clients can subscribe.


rest

  Autumn cleanup
We planned to move the cycles in a new calendar service.

Rest API. They are all static assets in the API, such as cards, decks, avatars, game types, etc.


rooms

  Autumn cleanup
We planned to not use Docker Compose anymore. The database used will be external.

The rooms service will manage discussions between players inside rooms, contained in subjects.


shop

  Autumn cleanup
We planned to not use Docker Compose anymore. The database used will be external.

  Protected repository
This repository is only available for The First Spine development team.

Manages all the purshases with real money. Works with the Stripe API.


solid-pancake

  Autumn cleanup
We planned to not use Docker Compose anymore. The database used will be external.

Solid Pancakes is an event-based consumer that tracks usages on the TFS products.


website

  Autumn cleanup
We planned to not use Docker Compose anymore. The database used will be external. Plus, the events entities will be moved in a new calendar service.

Website built with Sails.js, using all the other services to serve data to the players.
