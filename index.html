Building online board games
through open source technology

Infrastructure

Monitoring, deployment... all the server/docker-related things that empowers the TFS plateform.

boiler

Boiler is a Docker orchestration software built on top of nginx & docker-compose.

boiler-sign

Third-party certification service that generates private key / certificates in dotenv files.

monitoring

Services monitoring. Data collected with Loki, Promtail, cAdvisor, Prometheus. Monitored with Grafana.

flexible-pancake

Data catalog app to self-document data across services.

Services

The TFS plateform is powered by a Server-Oriented Architecture. These services are served by Boiler in containers.

arena

The arena service is the service that will manage all the games & wizzards accounts in the Arena products.

auth

Main authentification service. Manages registrations & token generation.

bots

Bot service to act like a human in the games. The bots should not be used for ranked games & tournaments.

messaging

Simple realtime messaging system. Contains an API to send message through sockets, where clients can subscribe. It's lighter than Apache Kafka because we have simple needs here.

rest

Rest API. They are all static assets in the API, such as cards, decks, avatars, game types, etc.

rooms

The rooms service will manage discussions between players inside rooms, contained in subjects.

shop

Manages all the purshases with real money. Works with the Stripe API.

solid-pancake

Solid Pancakes is an event-based consumer that tracks usages on the TFS products.

website

Website built with Sails.js, using all the other services to serve data to the players.

Tools & assets

All the tools & the assets we need to construct TFS apps.

sharer

sharer is a simple CLI to add shared assets accross apps using Git submodules. It also add required NPM dependencies in the package.json file of the shared assets.

flame

Data computing & analysis on-demand on CLIs served by Google Fire.

arena-shared

Shared & agostic resources for TFS platform. Served with the sharer tool. These resources are all related with the service Arena.

auth-shared

Shared & agostic resources for TFS platform. Served with the Sharer tool. These resources are all related with the service Auth.

env-shared

Shared & agostic resources for TFS platform. Served with the Sharer tool. Provides an `env` TS service to gather environment configurations through dotenv files or environment variables.

log-shared

Shared & agostic resources for TFS platform. Served with the Sharer tool. Provides a Nest logger service with `winston`.

messaging-shared

Shared & agostic resources for TFS platform. Served with the Sharer tool. These resources are all related with the service Messaging. Provides a Nest service to send messages to the service.

rest-shared

Shared & agostic resources for TFS platform. Served with the Sharer tool. These resources are all related with the service Rest.
