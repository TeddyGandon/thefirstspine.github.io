The Drifter’s Tales Worlds API is used to provide online features to the Drifter’s Tales game.

Endpoints documentation

This part describes the endpoints of the API.

POST /event

Register an event in the API. A message will be posted in the Messaging service, with a bounce of 60 seconds.

Authorization

This endpoint requires an access token under the Authorization: Bearer xxx.xxx.xxx header.

More information here: Endpoint protections
