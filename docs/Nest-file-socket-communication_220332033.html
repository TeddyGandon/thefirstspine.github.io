Tech : Nest file socket communication

Created by Teddy GANDON on Jan 31, 2021

In Arena, the current games are loaded in memory to ensure quick calculation and less I/O on databases. The Nest File Socket NPM extension allow any user to communicate with a working app.

Socket file format

The socket file named socketat the root of the app contains all the commands to call in the app context. It has this format:

{"name":"my-command", "args": ["first", "second"]}
{"name":"another-command", "args": ["third"]}

Available commands

close-game

Arguments:

  • gameId - the game ID
