The auth API is used to authenticate users to TFS apps.

Endpoints documentation

This part describes the endpoints of the API.

POST /api/v2/signup

Used to register a user in the auth service

Authorization

This endpoint is public.

Parameters

Parameter name

Type

Required ?

Description

email

string

yes

The email

password

string

yes

The password. Must have more than 8 characters.

Responses

Response code

Description

Example

400 Bad Request

The user already exists, or the provided parameters are malformed.

Can vary

200 Ok

Normal response

  {
    "user_id": 123
  }
