The Arena Shop API allows the clients to purchase some items (e.g. trade items for other).

Endpoints documentation

This part describes the endpoints of the API.

POST /shop/exchange

Trade items for other items.

Authorization

This endpoint requires an access token under the Authorization: Bearer xxx.xxx.xxx header.

More information here: Endpoint protections

Parameters

Parameter name

Type

Required ?

Description

String

yes

The shop item ID. Can be found in the REST API.
