Tech : Environment-based configuration of services

Created by Teddy GANDON, last modified on Jul 29, 2020

In the TFS Platform, every net service is configured with environment variables. These variables can be loaded with a dotenv file names .env at the root of the app, or can be assigned when a container is raised.
