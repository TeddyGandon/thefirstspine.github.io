The Arena Game API is a JSON-RPC 2.0 API used to interact with games & queues. See https://www.jsonrpc.org/specification for more informations about the JSPN-RPC 2.0 format.

Methods documentation

The following methods (exception of getGame and getQueur) requires an access token under the Authorization: Bearer xxx.xxx.xxx header.

More information here: Endpoint protections

joinQueue

Join a queue in the queue service for 60 seconds. This joining request should be refreshed with the refreshQueueAsk method.

Parameters

Parameter name

Type

Required ?

Description

key

string

yes

The key of the queue to join

destiny

string

yes

The destiny the user wants to play with. Can be found in the REST API.

origin

string

no

The origin the user wants to play witj. Can be found in the REST API.

style

string

no

The style the user wants to play with. The allowed styles can be found in the items of the wizard account.

cover

string

no

The cover the user wants to play with. The allowed covers can be found in the items of the wizard account.
